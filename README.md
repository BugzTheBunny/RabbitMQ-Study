### Setting a RabbitMQ instance
- Using Docker - `docker run -it --rm --name rabbitmq -p 5672:5672 -p 15672:15672 rabbitmq:3.9-management`

### Using RabbitMQ
- Using Python - `pip install pika`
- Using .Net  - `dotnet add package RabbitMQ.Client`
